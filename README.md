# KopДом (코르돔) — 프로토타입

러시아어권 이주민·유학생을 위한 대한민국 주거·생활 정보 플랫폼의 홈페이지 초안입니다.
`이행전략보고서_양성규-코르돔.pdf` 를 근거로 제작되었으며, **실제 서비스가 아닌 데모/프로토타입**입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속.

## 데모 계정

localStorage 기반 가짜 인증입니다 (실서비스 전환 시 반드시 교체 필요).

| 역할 | 이메일 | 비밀번호 |
| --- | --- | --- |
| 관리자 | admin@kordom.demo | admin1234 |
| 공인중개사 (승인됨) | agent@kordom.demo | agent1234 |
| 공인중개사 (승인대기) | agent2@kordom.demo | agent1234 |
| 일반회원 | seeker@kordom.demo | seeker1234 |

브라우저 개발자도구 콘솔에서 `localStorage.clear()` 실행 시 모든 데모 데이터가 초기화됩니다.

## 이 프로토타입의 한계 및 추후 개발 시 연동 지점

전문 개발팀이 인계받을 때 아래 지점을 실제 시스템으로 교체하면 됩니다.

1. **백엔드/DB** — `src/services/*.ts` 가 현재 `localStorage`를 사용합니다. 각 함수의 시그니처는 유지한 채 내부 구현만 실제 API 호출로 교체하면 됩니다 (`src/services/storage.ts` 참고).
2. **결제** — `src/services/payment.ts` 의 `createPayment()` 가 스텁입니다. 통역 동행료, 변호사 권리분석 보고서 비용, 공인중개사 매물 노출 광고비 등 결제가 필요한 지점에는 이미 `<PaymentStub />` 컴포넌트가 배치되어 있습니다 (`src/components/common/PaymentStub.tsx`). 토스페이먼츠/이니시스 등 PG SDK 연동 시 이 두 파일만 교체하면 됩니다.
3. **지도** — 현재 OpenStreetMap(Leaflet, API 키 불필요)을 사용 중입니다. 실제 서비스에서는 카카오맵/네이버맵으로 교체를 권장하며, `src/components/map/MapView.tsx` 한 파일만 교체하면 됩니다.
4. **인증** — 비밀번호가 평문으로 localStorage에 저장됩니다 (데모 전용). 실제 서비스에서는 서버 사이드 인증(해시, 세션/JWT 등)으로 교체해야 합니다.
5. **공인중개사 자격증 인증** — 현재는 파일명만 저장하고 관리자가 화면에서 승인/반려하는 흐름만 구현되어 있습니다. 실제로는 파일 업로드 스토리지 + 자격증 진위 확인(국가공간정보포털 등) 연동이 필요합니다.
6. **이용약관/개인정보처리방침** — `src/i18n/locales/*.json` 의 `terms`, `privacy` 섹션에 간단한 초안이 3개 언어로 작성되어 있습니다. 정식 출시 전 법무법인 KNK의 검토가 필요합니다 (약관 하단에 안내 문구 포함).
7. **번역 콘텐츠** — 매물/통역사/칼럼 시드 데이터는 `src/data/seed.ts` 에 있으며, 각 항목은 `{ko, en, ru}` 구조로 되어 있어 실제 데이터 입력 시 동일한 패턴을 따르면 됩니다.
